Detailed Description

Returns an isotropic flux with respect to a given Direction.

An isotropic flux is distributed as \( p_{\mu}(\mu) = 2\mu \) in \( [0, 1) \) where \( \mu \) is the cosine of the angle between the sampled Direction and the reference Direction.
