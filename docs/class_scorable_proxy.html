Detailed Description

A lightweight object for accumulating scores for a single history.

During a call to TransportMethod::Transport(), scores are produced by a single Particle object's random walk. However, we cannot immediately score an Estimator once that Particle has died. The Particle might have produced secondaries and those secondaries must complete their random walks too.
